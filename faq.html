🚗 주차 (관련 민원 178건)
Q1주차 위반 차량을 신고하고 싶어요. HOT
차량 번호와 위치를 관리사무소(☎ 02-931-8360) 또는 민원게시판에 알리면 방송·스티커 경고 조치합니다.

소화전·장애인 구역·이중주차 포함. 지속 점유 시 견인 가능. 킥보드·전기자전거 지하주차장 무단 주차도 동일하게 신고 가능합니다. 📊 약 89건
Q2전기차 충전구역에 일반 차량을 주차해도 되나요? HOT
'전기차 전용' 표시 구역은 충전 목적의 전기차만 주차 가능합니다.

충전 완료 후 14시간 이내 이동 의무(친환경자동차법). 미이동 시 과태료 대상. 📊 약 54건
Q3방문차량 무료 주차 시간은 얼마나 되나요?
월 150시간 무료 (2026.03.01 시행). 초과 시 30분당 500원 부과.

명절 연휴는 시간 차감 없음. 📊 약 35건
🏊 커뮤니티·시설 (관련 민원 121건)
Q4수영장·헬스장·독서실 등록은 어떻게 하나요? HOT
매월 지정 신청일에 바이비 앱 또는 커뮤니티센터 현장 등록.

수영장은 얼굴인식 별도 등록 필요. 독서실은 일일권 결제 필수. 사이트 오류 시 현장 방문 접수. 📊 71건
Q5커뮤니티 얼굴인식 등록 오류가 생겼어요.
커뮤니티센터 현장에서 재등록하거나 관리사무소에 방문 접수하세요.

앱 오류와 별개로 현장 등록이 가능합니다. 📊 50건
🛗 엘리베이터·하자 (관련 민원 133건)
Q6엘리베이터에서 이상한 소리·떨림이 납니다.
동·라인 번호와 증상(층, 소리 종류)을 관리사무소(☎ 02-931-8360)에 알리면 유지보수 업체가 즉시 점검합니다. 📊 66건
Q7집에 누수·결로·곰팡이가 생겼어요.
동·호수와 위치(천장·벽·화장실 등)를 관리사무소에 알리면 하자보수 업체를 연결해 드립니다.

입주 후 10년 이내 구조·방수 하자는 시공사 하자보수 대상입니다. 📊 67건
🧹 청소·위생 (관련 민원 50건)
Q8복도·주차장 청소가 안 되어 있어요.
위치와 상태를 민원게시판 또는 관리사무소에 신고해 주세요.

긴급 오염(오물·침수 등)은 경비실을 통해 즉시 조치 가능합니다. 📊 50건
🏗️ 인테리어·공사 (관련 민원 48건)
Q9인테리어 공사 중 엘리베이터 보양재가 오래 방치돼 있어요.
공사 완료 후 해당 세대가 관리사무소에 철거를 요청해야 합니다. 방치·분진 오염 시 동·라인 번호를 알려주시면 조치합니다. 📊 48건
🗑️ 쓰레기·폐기물 (관련 민원 39건)
Q10음식물 쓰레기 RFID 카드가 없으면 어떻게 되나요?
카드 없이 투입 시 건당 200원 벌금 부과. 카드는 관리사무소에서 발급합니다.

베란다 음식물 투척은 관리규약 위반입니다. 📊 약 25건
Q11대형폐기물(소파·냉장고)은 어떻게 버리나요?
서대문구 다산콜 ☎ 02-391-7171 또는 minwon.seoul.go.kr 신청 → 납부 → 접수번호 경비실 통보 후 배출.

스티커 없이 배출 시 과태료 처분 대상입니다. 📊 약 14건
🌳 조경·외부 (관련 민원 35건)
Q12고사목·잡초·나무 전지 정비를 요청하고 싶어요.
위치를 명시하여 민원게시판에 신고해 주세요. 조경 전지는 봄·가을 연 1~2회 정기 시행됩니다. 📊 35건
🛡️ 경비·보안·시설 (관련 민원 63건)
Q13외부인이 단지 내에 무단 출입하고 있어요.
경비실(☎ 02-931-8360)에 즉시 신고. 24시간 운영 중. CCTV 확인 필요 시 관리사무소에 요청하세요. 📊 33건
Q14공용시설(운동기구·벤치·조명)이 파손되었어요.
파손 위치와 상태를 민원게시판에 신고해 주세요. 사진 첨부 시 빠른 처리에 도움이 됩니다.

안전 위험이 있는 경우 경비실을 통해 즉시 접수하세요. 📊 30건
🚬 흡연·냄새 (관련 민원 26건)
Q15화장실·안방에서 담배 냄새가 올라옵니다.
배관을 통한 인근 세대 흡연이 주요 원인입니다. 관리사무소 신고 시 금연 방송·주의 안내문 발송 가능.

환기구 역류 방지 장치 점검도 요청하세요. 지정 흡연구역 외 흡연 적발 시 과태료. 📊 26건
💰 관리비 (관련 민원 다수)
Q16관리비 고지서 상세 내역을 어디서 확인하나요?
dmcpvx.com 로그인 → '관리비 조회' 메뉴에서 월별 상세 내역 확인.

공동관리비 차감액은 단지 공용 수익(광고·임대) 환급분으로 매월 달라집니다.
🐾 반려동물 (관련 민원 12건)
Q17반려동물 공용공간 이용 규정이 어떻게 되나요?
공용공간에서는 목줄 착용 필수이며 배변 즉시 처리 의무.

엘리베이터·복도에서 목줄 미착용 시 관리규약 위반으로 민원 접수 대상입니다. 📊 12건
🔥 안전시설 (관련 민원 10건)
Q18소화기·비상구 등 안전시설 이상을 발견했어요.
위치를 명시하여 관리사무소 또는 경비실에 즉시 신고. 긴급 상황 시 119.

비상구 앞 적치물, 소화기 위치 이탈 등은 빠른 신고가 중요합니다. 📊 10건
📦 택배·무인함 (관련 민원 8건)
Q19택배 무인함 비밀번호를 못 받았거나 분실됐어요.
문자 미수신 시 관리사무소(☎ 02-931-8360)에 문의하시면 확인해 드립니다.

보관 기간 초과 시 반송 처리될 수 있으므로 빠른 수령을 권장합니다. 📊 8건
🔊 층간소음 (관련 민원 다수)
Q20층간소음 민원 절차가 어떻게 되나요? HOT
타워동은 구조 특성상 발생 세대 특정이 어렵습니다. 소음 시간대·위치(천장·벽 방향)를 기록해 관리사무소에 제출하면 층간소음 측정 조사를 요청할 수 있습니다.

지속 분쟁은 층간소음이웃사이센터(☎ 1661-2642)를 통한 공식 조정도 가능합니다.
